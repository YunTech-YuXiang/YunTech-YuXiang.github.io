1.安裝LAMP環境:

安裝 Apache Web 主機，打上yum install httpd即可安裝，啟用systemctl start httpd.service，檢查狀態systemctl status httpd.service，如果出現綠色的active(running)表示安裝成功。

安裝PHP，打上yum install php即可安裝，要再重新啟用一次systemctl restart httpd.service，即完成安裝。接著要測試PHP程式，將工作目錄轉換至cd /var/www/html，打上gedit info.php，打開文字編輯器輸入<?php>phpinfo();<?>這個程式可以顯示版本號碼，在到Linux系統中，防火牆裡新增一個埠號80，即完成。

最後查詢虛擬機的IP，並且在client端執行網頁，輸入IP位址加上info.php，呈現下圖，表示測試程式成功。

2.安裝 phpMyAdmin 資料庫管理

打上yum install sqld-server即可安裝，啟用systemctl start mysqld.service，再打上systemctl enable mysql.service可以在下次開機時自動執行此服務，切換目錄cd /etc/httpd/conf.d，開啟文字編輯gedit phpMyAdmin.conf，並且在第一個Directory下的在後面加入Require all granted，最後再重啟用一次systemctl restart httpd.service，然後設定密碼mysq_secure_installation，設定好即成功。

回到client端開啟瀏覽器打上虛擬機的IP位址，如http://192.168.236.128/phpmyadmin，呈現如下圖。

3.一開始先打systemctl status httpd.service查看狀態，接著切換目錄到cd /var/www/html進入預設網頁位置，在打上gedit index.html用以修改首頁內容，在文字編輯器中打上以下內容。
